## Processing Guide

When processing inbox items, ask:
1. **Is it actionable?**
   - Yes → Create task in today's log or work item
   - No → Delete or move to reference

2. **Is it a single task?**
   - Yes → Add to daily log
   - No → Create work item and breakdown

3. **Can AI do it?**
   - Yes → Mark as [AI] task
   - No → Mark as [P] task

4. **When should it happen?**
   - Today → Add to today's log
   - Specific date → Add to that day's log
   - Someday → Add to someday.md (create if needed)
